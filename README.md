# 番茄时钟 (Pomodoro Timer)

一个基于 Electron 的桌面番茄时钟应用。

## 功能

- **三模式切换**：长计时(25分钟)、短计时(5分钟)、计时器(正计时)
- **自动休息**：长/短计时结束后自动进入 5 分钟休息
- **系统通知**：工作/休息结束时弹出桌面通知
- **天蓝色界面**：清爽的天蓝色渐变主题，全中文

## 运行

```bash
npm install
npm start
```
